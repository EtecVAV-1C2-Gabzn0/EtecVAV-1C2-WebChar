Este é um Quiz interativo em Python desenvolvido para nota de TPA sobre a plataforma Arduino, seus componentes de hardware, sintaxe de programação (baseada em C/C++), e protocolos de comunicação.

Funcionalidades
  ◦Banco de Perguntas Extenso: 
      ◦ Contém 50 perguntas abrangentes sobre diversos tópicos do Arduino.
  ◦Seleção Aleatória de Perguntas: 
      ◦ A cada rodada, o quiz seleciona 20 perguntas aleatoriamente para garantir uma experiência de aprendizado variada.
  ◦ Embaralhamento de Alternativas: 
      ◦ A ordem das alternativas (A, B, C, D, E) é randomizada para evitar memorização da posição.
  ◦ Contagem de Pontuação: 
      ◦ Cada acerto vale 10 pontos, com exibição do resultado final (pontuação total e percentual de acerto).
  ◦ Interface de Menu: 
      ◦ Navegação simples para iniciar o quiz, ver as regras ou encerrar o programa.
